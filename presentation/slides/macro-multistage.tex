\begin{frame}[fragile]{Multi-stage programming}

  \begin{block}{}
    The runtime can be bundled with the Scala compiler itself. This is know as \textbf{multi-stage programming}, and can be used to implement a JIT compiler:
  \end{block}

  \vfill

\begin{scalacode}
given Compiler = Compiler.make(getClass.getClassLoader)

staging.run:
  '{ println("Hello world!") }
\end{scalacode}

\end{frame}
